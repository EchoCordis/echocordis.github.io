SPELUNKA-BOT

A recreation of the NES game Bomberman (1985), built using the Unity engine.

Role

Context

Date
UI Designer / Programmer University Project 2021

OVERVIEW
Spelunka-Bot was developed during a group assignment that was structured similarily to a week-long game jam. As a group, we were provided a design document from another group of students where we had to recreate a retro game based of the specifications provided. My group was tasked with recreating the NES game, Bomberman.

This project was developed in partnership with Roy Fu, Padriac Heaton, Rhys Vincent and Long Vo.

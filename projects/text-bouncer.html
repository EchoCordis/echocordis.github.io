TEXT BOUNCER

An interactive program that takes user text input and bounces the text around the screen while playing musical notes.

Role

Context

Date
Creator University Project 2020

OVERVIEW
The Text Bouncer was developed for an university project which required transforming user input into visual and audio output. This program takes the user's text input and draws the text on screen. As the text bounces around the screen, different sound effects play on random. The user can also adjust the font and colour of the text.

The Text Bouncer was developed using the Processing IDE and Java.
